# 🧬 HYA28 Toolkit

**Your one-stop bioinformatics toolkit - turning messy data into beautiful science.**

A free, browser-based collection of bioinformatics and data visualization tools. No installations, no sign-ups, no data uploads to any server - everything runs right in your browser.

🔗 **Try it now:** [https://harshapurva.github.io/HYA28-toolkit/](https://harshapurva.github.io/HYA28-toolkit/)

---

## 🛠️ Available Tools

### 📊 CSV Data Visualizer

Upload any CSV file, select columns, filter values and instantly generate interactive **pie** or **doughnut charts**. Export your filtered data as CSV or download **publication-ready PNG** images.

### 🧬 Sequence Analyzer

Paste a DNA or protein sequence and the tool **auto-detects the type**. Get GC content, codon usage, reverse complement, 6-frame translation, amino acid frequency and molecular weight - all in one click. Supports **multi-FASTA** input.

### 🔍 Interactive ORF Finder

Scan all **6 reading frames** for open reading frames. Visualize ORFs in a mini genome browser with clickable bars, protein translations and adjustable minimum length. Supports **FASTA file upload**.
